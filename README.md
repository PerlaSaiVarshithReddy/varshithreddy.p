# Live News API Example

This is a minimal Node.js example that fetches live news using a news API key.

## Setup

1. Replace `YOUR_API_KEY_HERE` in `fetchNews.js` with your real API key.
2. Run the script:

```bash
cd /Users/saivarshithreddyperla/Desktop/livenewsapi
node fetchNews.js
```

## Notes

- This example uses `https://newsapi.org/v2/top-headlines`.
- If you want a different endpoint or provider, update `endpoint` in `fetchNews.js`.
- You can change the query parameters in `fetchNews.js` to fetch different categories, search terms, or countries.
